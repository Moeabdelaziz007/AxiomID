# AxiomID LangChain Integration

Official LangChain integration for AxiomID — enabling AI agents to use decentralized identity, trust scores, and Soul Gate enforcement.

## Features

- **DID Verification**: Verify `did:axiom` identities for agents
- **Soul Gate Enforcement**: Check ethical gates before task execution
- **Trust Score Context**: Fetch and display trust scores as agent context
- **Attestation Issuance**: Issue verifiable attestations from within agents
- **Streaming Support**: Full async/streaming support for real-time agents
- **Delegation Chains**: Multi-agent delegation (A → B → C) with identity verification
